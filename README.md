# VAR Cars

VAR Cars is a school-based project developed for academic purposes. This system
was created as part of a student final project to demonstrate web application
development concepts such as user registration with email verification, login
sessions, a shopping cart, checkout, and an admin dashboard in a luxury car
dealership themed platform.

## Purpose

This project is intended for school use and educational presentation. It was
built as the final project for **Applications Development and Emerging
Technologies**. It serves as a sample application for learning, demonstration,
and project evaluation. No real vehicles are sold and no real payments are
processed.
